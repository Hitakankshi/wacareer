# **App Name**: Royal Ascent Careers

## Core Features:

- Job and Internship Listings: Display current job and internship postings with search and filtering options. Auto email notification on application.
- Online Course Platform: Allow users to browse, purchase, and access online courses. Support course previews and downloadable materials.
- Student Dashboard: Enable registered users to access purchased courses, track progress, and generate certificates upon completion. Students can interact with chat-based support.
- Employee Dashboard: Provide a platform for employees to view assigned tasks, record attendance, and access payroll information.
- Admin Dashboard: Allow administrators to manage job postings, internships, courses, student data, employee data, applications, and overall system analytics.
- AI-Powered Resume Parsing: Use AI to parse and extract information from resumes uploaded during job applications, acting as a tool for populating application fields automatically.
- User Authentication: Secure user authentication with email/password, Google login, and phone number OTP login, all managed through Firebase.

## Style Guidelines:

- Primary color: Deep Gold (#D4AF37) to convey luxury and richness.
- Background color: Matte Black (#0A0A0A) to provide a sophisticated backdrop.
- Accent color: Soft White (#FFFFFF) to ensure text legibility and contrast.
- Body and headline font: 'Playfair' (serif) for headlines and limited text chunks; use 'PT Sans' (sans-serif) for body text where needed.
- Use premium, custom-designed icons with a gold gradient to match the luxury theme.
- Implement a minimalist, spacious layout with a focus on high-quality imagery and clear typography to enhance the user experience.
- Employ subtle, elegant animations (e.g., smooth transitions, gold particle effects) to add a touch of luxury and interactivity.
